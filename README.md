## Сценарии перехода на страницу курса "Тестировщик ПО":
###   Сценарий 1: запись на обучение профессии "Тестировщик ПО", через "Каталог курсов":
1. Открыть браузер
2. Перейти по [ссылке](https://netology.ru/)
3. Нажать "Каталог курсов"
4. В меню выбора "Направления обучения", выбрать "Программирование"
5. В списке курсов по направлению «Программирование» выбрать курс "Тестировщик ПО"
##### Ожидаемый результат: открыта страница курса "Тестировщик"

###   Сценарий 2: запись на обучение профессии "Тестировщик ПО", через меню поиска в "Каталог курсов":
1. Открыть браузер
2. Перейти по [ссылке](https://netology.ru/)
3. Нажать "Каталог курсов"
4. В строке ввода "Какой курс вам нужен" ввести "тест"
5. В появившемся окне "Все курсы по запросу "тест", выбрать курс "Тестировщик ПО"
##### Ожидаемый результат: открыта страница курса "Тестировщик"

###   Сценарий 3: запись на обучение профессии "Тестировщик ПО", через форму "Направления обучения":

2. Перейти по [ссылке](https://netology.ru/)
3. В форме "Направления обучения" выбрать "Программирование"
4. В списке курсов по направлению "Программирование" выбрать курс "Тестировщик ПО"
##### Ожидаемый результат: открыта страница курса "Тестировщик"

###  Сценарий 4: запись на обучение профессии "Тестировщик ПО", через форму "Направления обучения", используя кнопку "Полный каталог":

3. В форме "Направления обучения" нажать кнопку "Полный каталог"

## Сценарии заполнения формы "Запишитесь на курс":

###   Сценарий 10: ввод валидных значений не авторизированного пользователя в форме "Запишитесь на курс":

2. Ввести в поле "Имя" валидное значение на кириллице или латинице
